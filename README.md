# Restaurant Data Analysis Project

This project was completed as part of the Data Analysis Internship Program by Cognifyz Technologies.

## Project Overview

The project focuses on analyzing restaurant data to discover insights related to:

- Popular cuisines
- City-wise restaurant distribution
- Price range analysis
- Online delivery trends
- Restaurant ratings
- Cuisine combinations
- Geographic restaurant distribution
- Restaurant chain analysis

---

# Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Folium
- Jupyter Notebook

---

# Project Structure

## Level 1 Tasks
1. Top Cuisines Analysis
2. City Analysis
3. Price Range Distribution
4. Online Delivery Analysis

## Level 2 Tasks
1. Restaurant Ratings Analysis
2. Cuisine Combination Analysis
3. Geographic Analysis
4. Restaurant Chains Analysis

---

# Key Insights

- Identified the most popular cuisines
- Found cities with the highest restaurant concentration
- Analyzed customer ratings and votes
- Visualized restaurant locations using maps
- Compared restaurant chains by ratings and popularity
